# AllForCinema

Movie discovery and streaming platform built with React, Supabase, and Tailwind CSS.

## Features

- Browse movies by categories
- Create and manage playlists
- User authentication
- Admin panel for content management
- Modern dark UI

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS
- Supabase (Auth + Database)
- React Router

## Getting Started

```bash
npm install
npm run dev
```

## Environment Variables

Create `.env` with:
```
VITE_SUPABASE_URL=your-supabase-url
VITE_SUPABASE_ANON_KEY=your-anon-key
```
